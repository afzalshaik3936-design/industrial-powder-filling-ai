# 📌 Project Overview

This project demonstrates how Machine Learning can be applied in pharmaceutical manufacturing to predict tablet weight using manufacturing process parameters.

The application simulates an industrial decision-support dashboard where users can adjust production parameters, generate predictions, and analyse manufacturing data through interactive visualizations.

The project combines data analysis, regression modelling, dashboard development, and deployment into a single end-to-end application.

# 📊 Input Features

The prediction model uses eight manufacturing variables:

| Feature | Description |
|----------|-------------|
| Powder Density | Density of powder material |
| Moisture Content | Moisture percentage |
| Particle Size | Powder particle size |
| Compression Force | Tablet compression pressure |
| Machine Speed | Production speed |
| Fill Depth | Filling depth |
| Ambient Temperature | Production temperature |
| Relative Humidity | Manufacturing humidity |

# 💻 Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Industrial-Powder-Filling-AI.git
```

Move into the project

```bash
cd Industrial-Powder-Filling-AI
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
